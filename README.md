# 💡 Startup Idea Board
A full-stack web application for submitting, browsing, and discovering 
startup ideas — built from scratch with no prior coding experience.

🔗 **[View Live App](https://startup-idea-board-4k8p.vercel.app/)**

---

## 📌 Project Overview
Built during an AI-Assisted App Development Bootcamp as my first-ever 
coding project. The app allows users to submit startup ideas, browse 
ideas posted by others, and interact with a real-time database — all 
through a clean, responsive interface.

This project demonstrates my ability to learn new technical skills 
rapidly and independently, going from zero coding experience to a 
fully deployed web application.

---

## 🛠️ Tech Stack
| Layer | Technology |
|---|---|
| **Frontend** | Next.js, React, TypeScript |
| **Database** | InstantDB (real-time) |
| **Styling** | Tailwind CSS |
| **Deployment** | Vercel |
| **Version Control** | GitHub |

---

## ✨ Features
- Submit new startup ideas with title and description
- Browse all submitted ideas in real time
- Responsive design — works on desktop and mobile
- Live deployment with continuous integration via Vercel
